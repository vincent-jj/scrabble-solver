Notice d'utilisation du scrabble

Les informations écrites ci-dessous sont IMPORTANTES pour une bonne utilisation du programme !

/!\ Le dictionnaire doit se trouver dans le même dossier que l'executable ! /!\

#######################################################################

// Utilisation du Makefile \\

Pour compiler le programme : make

Pour supprimer les fichiers .o : make clean

Pour supprimer l'executable scrabble : make supraclean

#######################################################################

// Options disponibles pour le scrabble \\

Deux arguments au maximum peuvent être passés : '-n' et/ou '-m', l'ordre n'a pas d'importance.

'-n' : Cet argument permet d'activer la liste des 'n' meilleurs mots

'-m' : Cet argument permet de passer en mode tirage manuel

#######################################################################

// Fonctionnement du tirage manuel \\

Lettres acceptés : Minuscules et majuscules

Jokers : Les jokers sont représentés par le caractère '@' (ils doivent être placé en cellule 0 et/ou 1, en cas d'oubli le programme le rappelle)

Autres caractères : Tout autre caractère donnera lieu à un message disant : 'Lettre non-existente'

Pour ne pas mettre de caractère : Il suffit d'envoyer un espace

Pour ne pas écraser une lettre déjà présente : Il suffit d'appuyer sur entrée

Pour supprimer un caractère : La touche 'suppr' permet de supprimer le caractère précédent

#######################################################################

// Fonctionnement de la liste des 'n' meilleurs mot \\

Pour placer un mot de la liste, il suffit d'entrer le numéro associé au mot

#######################################################################

// IMPORTANT \\

Il se peut que pour certains tours de jeu le programme mette beaucoup de temps avant d'envoyer un résultat, notamment lorsque l'option '-n' est active. Le temps peut aller de quelques secondes à 5-10 minutes (trés rare) il ne faut donc surtout pas couper le processus ! 

Entre chaque tour de jeu il faut appuyer sur 'entrée' pour passer au tour suivant (c'est précisé lors du fonctionnement du programme) 

#######################################################################
